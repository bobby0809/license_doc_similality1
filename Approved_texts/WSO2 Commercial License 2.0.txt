JProfiler
All contributors who have signed the above CLA with WSO2 are eligible to attain an open source license key for JProfiler to be used specifically for WSO2 open source project development in Java. JProfiler is all-in-one Java profiler with an intuitive GUI that helps you find performance bottlenecks, pin down memory leaks and resolve threading issues.
